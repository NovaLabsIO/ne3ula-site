# NE3ULA Core — Source of Truth

This document defines the canonical architecture, philosophy, and development rules for the NE3ULA core website.

It exists to maintain continuity across:
- development sessions
- AI collaborators
- human collaborators
- future expansions of the NE3ULA ecosystem

If conflicts arise between documentation and code, this document should be updated to reflect the intended architecture.

---

# 1. Project Overview

NE3ULA is the philosophical and creative container for a larger system of tools, identity layers, and creative outputs.

The website serves three primary purposes:

1. Introduce the NE3ULA philosophy
2. Provide an entry point into music and creative output
3. Guide users into the AXZIO interface system

The website is not a conventional marketing site.

It is structured as a threshold → hub → path system.

# 7. Development Rules

When expanding the site:

1. Extend existing architecture before creating new pages
2. Preserve philosophical consistency
3. Keep copy minimal and intentional
4. Maintain dark/cosmic design language
5. Avoid unnecessary complexity

The site should feel like a quiet gateway, not a loud product website.
