# NE3ULA Philosophy — Site Copy Planning

## Overview

This document contains the planned public-facing copy for the NE3ULA philosophy system. It defines page hierarchy, navigation flow, hero language, and call-to-action sequences.

**Status**: Planning document. Code implementation is deferred.

**Architecture note**: Philosophy and myth are distinct layers. Philosophy defines the core mission, values, principles, and objectives of NE3ULA. Myth is the symbolic narrative embodiment of those truths. They are related and will be linked on the site, but should not be collapsed into one another. Implementation will preserve this distinction.

- **Sovereignty** = why the individual remains above the system

## Navigation & CTA Flow

The page sequence follows a deliberate pedagogical path:

1. **Doctrine** introduces what NE3ULA is and its purpose
2. **Laws** provides the condensed principles in memorable form
3. **Becoming** reframes the relevance to the individual human journey
4. **Functions** explains the operational mechanism
5. **Sovereignty** establishes the ethical safeguard (individual remains above system)
6. **Path** describes how doctrine becomes lived practice
7. **Principles** serves as a reference layer for those seeking deeper logic

Each page provides forward navigation to the next, plus secondary navigation to related pages.

---

## Notes for Implementation

- Preserve the strong hero language and page-purpose distinctions through implementation
- Maintain the clear CTA flow and button labels
- Philosophy pages should be visually and navigationally distinct from myth pages, though linked
- Responsive design should maintain copy hierarchy and readability
- The sequence should feel like a guided path, not a random collection of pages
